## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Create a local env file (recommended: copy [.env.example](.env.example) to `.env.local`) and set `VITE_GEMINI_API_KEY` (plus the `VITE_FIREBASE_*` keys)
3. Run the app:
   `npm run dev`
